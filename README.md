Hotel In-Booking System
Project Overview
The Hotel Inbuilt Booking System is a digital ordering platform designed to streamline the restaurant dining experience. Customers can log in with their name, table number, and contact details to access a personalized dashboard displaying available menus and dishes.

Features
Customer Dashboard: Browse the menu, add dishes to the order list, review, and submit orders.
Manager Dashboard: View customer details, selected dishes, quantities, and total bill. Confirm orders with real-time notifications sent to customers.
End-to-End Functionality: Implements smooth user flows, efficient interfaces, and real-time order management.

Technologies Used
Frontend: React.j
Backend: Node.js / Express.js
Database: MongoDB (or your database)

How It Works
Customer logs in with name, table number, and contact number.
Customer browses menu, selects dishes, and submits order.
Manager receives the order, verifies it, and confirms.

Key Skills Demonstrated
Full-stack development
Frontend interface design
Real-time data handling
User-friendly experience design
